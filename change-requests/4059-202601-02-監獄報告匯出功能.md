## 變更需求

變更原因及內容說明：

配合矯正機關（監獄）檢驗報告交付需求，需開發專用的 Excel 報告匯出功能，產出符合監獄格式要求的檢驗報告。

**變更原因：**
1. 監獄單位要求以特定 Excel 格式交付檢驗報告
2. 現有報告匯出功能無法滿足監獄格式需求（民國年、特殊欄位配置）
3. 檢驗結果值需經過轉換處理（如 pTestResult 函數），非原始數據直接匯出
4. 異常值需以視覺方式標示，便於監獄醫護人員快速辨識

**變更內容：**
1. 新增 PrisonReport 匯出模組
2. 自訂 Excel 格式排版（監獄專用欄位配置）
3. 結果值轉換：整合 pTestResult 函數，將原始數據轉換為報告呈現值
4. 日期格式處理：全面轉換為民國年格式（如 114/01/20）
5. 異常值標示：異常結果以紅色文字呈現
6. 報告匯出功能增強：支援批次匯出

**影響範圍：** lisweb 系統報告匯出模組

申請人員：紀伯喬
