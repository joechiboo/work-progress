## 變更需求

變更原因及內容說明：

LDT（Laboratory Developed Test）報告需整合基因相關學術文獻，以提升報告的學術參考價值，協助臨床醫師解讀基因檢測結果。

**變更原因：**
1. LDT 報告目前缺乏基因相關文獻引用，醫師需自行查詢
2. 手動查詢 PubMed 文獻耗時，且無法確保引用最新研究
3. 需建立自動化文獻查詢 API，供 LDT 報告產生器自動嵌入

**變更內容：**
1. 開發 Gene Reference API（.NET Web API）
   - 整合 NCBI ELink API 進行基因文獻查詢
   - 當 ELink 失敗時自動 fallback 至 PubMed 直接搜尋
   - 批次文獻查詢，每基因限制 Top 3 篇論文
2. LDT 報告產生器（LDTGenerator）整合
   - 自動查詢報告中涉及之基因文獻
   - 將查詢結果嵌入報告參考文獻區段
   - 文獻格式統一化（作者、期刊、年份）
3. IIS 部署配置
4. 部署文件與 README API 端點說明撰寫

**影響範圍：** lisweb LDTGenerator 模組、新增 gene-reference-api 服務

申請人員：紀伯喬

## 變更評估

**影響分析：**
- 影響模組：lisweb - LDTGenerator（報告產生器）

- 備份作業：部署前備份現有 LDTGenerator DLL
- 變更失敗處理：回復至前一版本 DLL，移除 API 站台
