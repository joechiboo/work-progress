## 變更需求

變更原因及內容說明：

為配合公司資安演練需求，需開發社交工程演練系統，模擬釣魚郵件發送與員工點擊行為追蹤，以評估組織資安意識並作為教育訓練依據。

**變更原因：**
1. 公司需定期執行社交工程演練，現無自有演練工具
2. 外部演練服務成本高，且無法客製化演練情境
3. 需追蹤員工點擊釣魚連結的行為數據，作為資安教育訓練參考

**變更內容：**
1. 開發 .NET 8 Web API 後端系統
2. 實作 Email 模擬發送功能（SMTP 整合）
3. 建立點擊追蹤機制，記錄員工開信與點擊行為
4. 實作加解密功能，保護演練資料與追蹤連結
5. 前端 UI 開發（演練管理、結果統計儀表板）
6. SQL Server 資料庫設計與建置
7. 撰寫部署文件與資料庫結構文件

**影響範圍：** 新增獨立系統，不影響現有系統

申請人員：紀伯喬
